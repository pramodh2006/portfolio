## ✨ Features

- 🌌 **Animated Three.js Star / Particle Background** — Immersive, GPU-rendered starfield that reacts to the scene in real time
- 🔷 **3D Wireframe Geometries** — Rotating Three.js meshes adding depth and visual interest to the hero section
- ⌨️ **Typewriter Hero Animation** — Smooth, looping typewriter effect that cycles through roles and titles
- 🎞️ **Scroll Reveal Animations** — Elements fade and slide into view as you scroll through each section
- 🃏 **3D Card Tilt Effect** — Perspective-aware tilt on project cards for a tactile, interactive feel
- 🗂️ **Interactive Experience Tabs** — Clean tabbed interface to navigate work and academic experience
- 🔦 **Active Section Nav Highlighting** — Navigation links update dynamically based on the current scroll position
- 📱 **Fully Responsive Design** — Pixel-perfect layout across desktop, tablet, and mobile viewports

## ⚙️ Getting Started

Clone the repository and open it locally — no build step, no package manager, no config required.

```bash
# 1. Clone the repo
git clone https://github.com/pramodh2006/portfolio.git

# 2. Navigate into the project
cd portfolio

# 3a. Open directly in your browser
open index.html

# 3b. Or use Live Server in VS Code for hot-reload
#     Right-click index.html → "Open with Live Server"
```

> **Tip:** Using VS Code's [Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) is recommended for the best local development experience.
